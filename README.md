#  How to Install jupyter notebook.

Install jupyter-notebook on linux.
- run 
```bash
1.$ python3 -p pip install --upgrade pip
```
```bash
2. $ pip install jupyter
```

```bash
3. $ sudo apt install jupyter -y
 ```
